A console-based Python application that fetches real-time weather details using a public weather API.

## Features
- User enters city name
- Displays temperature and weather condition
- Uses API calls and JSON parsing

## Technologies Used
- Python
- Requests library
- OpenWeatherMap API

## API Key Setup

This project uses the OpenWeatherMap API.

To run the application:
1. Create a free API key from https://openweathermap.org/api
2. Open `weather_app.py`
3. Replace:
   api_key = "YOUR_API_KEY_HERE"
   with your own API key
4. Run the program
